# McKinley & Rice Node.js Task Solution

## How to run locally

```
$ cd into the project dir
$ npm install
$ npm start

The server runs on port 5000.
```

## APIs

```
1. POST `/api/register`

body = {email, password}

returns error or success message

2. POST `/api/login`

body = {email, password}

returns error or JWT Auth Token
```
